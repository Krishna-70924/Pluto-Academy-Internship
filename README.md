# 📺 Netflix Content Analysis & Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Netflix Titles dataset to understand the platform's content library. The analysis covers data cleaning, preprocessing, visualization, and extracting meaningful insights about Netflix's movies and TV shows.

The project is implemented using **Python** in a **Jupyter Notebook** and demonstrates the complete data analysis workflow.

## 🎯 Objectives

* Explore the structure and quality of the Netflix dataset.
* Clean and preprocess missing and inconsistent data.
* Perform exploratory data analysis (EDA).
* Visualize important trends using charts and graphs.
* Generate business insights from the data.

* **Total Records:** 8,807

## 🧹 Data Cleaning

The following preprocessing steps were performed:

* Removed unnecessary columns.
* Handled missing values in:

  * Director
  * Cast
  * Country
* Converted `date_added` into datetime format.
* Extracted numeric duration values.
* Removed remaining incomplete records.
* Standardized text columns.
* Verified that the dataset contained no duplicate rows.

## 📊 Exploratory Data Analysis

The notebook answers several analytical questions, including:

* Distribution of Movies vs TV Shows
* Top countries producing Netflix content
* Most common content ratings
* Content added each year
* Movie duration trends
* Genre distribution
* Correlation between release year and year added
* Heatmap and statistical analysis


## 📈 Key Insights

* Movies make up nearly **70%** of Netflix's catalog.
* TV-MA is the most common content rating.
* Most modern movies are around **90–100 minutes** long.
* Netflix experienced its highest content additions in **2019**.
* Netflix regularly adds both new releases and older classic titles.
